ThinkAdmin v1.0
===============

基于ThinkPHP5.1 感谢ThinkPHP官方的努力祝ThinkPHP越来越好,感谢LayUI让写html变得更加简单
本框架为轻量级后台管理系统,只提供最基础功能,更多功能需根据实际业务由个人进一步开发
核心功能:

 + 后台菜单管理
 + Auth权限检查

需要注意的两个基类:

 + Adminbase 严格的权限检查
 + NotAuth 需要登陆但是不需要权限检查
 
实际开发中有许多应用场景需要权限检查,也有特殊的场景只登陆不要做权限检查,这两个基类可以满足需要
 
> ThinkPHP5的运行环境要求PHP5.6以上。
